\begin{abstract}
本文研究 Beer Game 供应链环境中的单智能体库存订货控制问题。当前代码中，一个企业作为学习智能体，其余企业采用简单的 base-stock 启发式策略。本文基于当前 \texttt{results/} 目录中的最新完整运行结果和刷新后的报告图表，对 DQN、Double DQN、PPO 以及高维动作空间下的 High-Dim DQN、High-Dim Double DQN 和 High-Dim PPO 进行比较。在标准一维订货动作空间中，Double DQN 取得最高平均测试收益 631.92，PPO 则取得最低收益标准差 39.30 和最高需求满足率 0.934。高维动作空间中，High-Dim Double DQN 平均收益最高，为 445.50；新增的 High-Dim PPO 平均收益为 406.50，收益波动低于两个高维价值函数方法。当前结果说明，标准动作策略整体仍强于高维策略；高维订货建模对算法选择和优化细节都较敏感。
\end{abstract}

\section{引言}

Beer Game 是供应链管理中的经典动态决策问题。多个企业按串行结构组成供应链，下游企业面对外部随机需求，上游企业根据下游订单进行补货。由于订单传递、库存约束、持有成本和缺货惩罚共同作用，局部看似合理的订货行为可能带来库存积压、缺货损失以及供应链波动放大。

本项目最初实现了 DQN baseline。当前实验框架进一步扩展为四个方向：第一，引入 Double DQN，通过分离动作选择和目标值评估来缓解 DQN 的过估计问题；第二，引入 PPO，使用 clipped policy gradient 直接优化随机策略；第三，将一维订货动作扩展为三维订单向量，用于模拟多渠道或多供应商订货分配；第四，将 PPO 也应用到高维订单空间中，把枚举后的向量订单作为 categorical policy 的离散动作。

本文根据当前实验输出重新撰写报告，不再沿用旧图和旧命令。重点回答以下问题：
\begin{itemize}
    \item 标准动作空间下哪个算法的收益和稳定性最好？
    \item 当前结果中 Double DQN 是否显著优于 DQN？
    \item 高维动作空间相比一维订货动作带来了多大训练难度，PPO 是否能改善该问题？
    \item 后续训练和实验管理应该如何调整？
\end{itemize}

\section{环境与目标}

\subsection{Beer Game 模型}

实验环境包含 $N=3$ 个企业。企业 $0$ 最接近外部市场，企业 $1$ 是所有实验中的学习企业，企业 $2$ 是上游供应企业。每个 episode 最多运行 100 个时间步。外部市场需求服从泊松分布：
\begin{equation}
D_t \sim \operatorname{Poisson}(10).
\end{equation}

在标准动作空间中，学习企业每个时间步选择一个离散订货量：
\begin{equation}
a_t \in \{1,2,\ldots,20\}.
\end{equation}
局部观测包括该企业的订货量、已满足需求和库存水平。库存更新遵循 Beer Game 中的库存平衡关系：
\begin{equation}
I_{t+1}=I_t+q_t-d_t,
\end{equation}
其中 $q_t$ 表示环境转移中记录的补货或订货量，$d_t$ 表示实际满足的需求。

\subsection{奖励函数}

学习目标是最大化企业在整个 episode 中的累计利润。即时奖励由销售收入、采购成本、库存持有成本和缺货惩罚构成：
\begin{equation}
r_t=p_i d_t-p_{i+1}q_t-hI_t-c\max(0,D_t-d_t).
\end{equation}
当前实验使用价格 $[10,9,8]$，库存持有成本 $h=0.5$，缺货惩罚 $c=2.0$，初始库存为 100，每个 episode 最多 100 步。

\subsection{非学习企业策略}

非学习企业采用 base-stock 启发式策略。相比随机策略，该设置可以降低环境噪声，使算法之间的差异更容易解释。Base-stock 策略根据目标库存和当前库存之间的差距确定订货量，并裁剪到可行动作范围内。

\section{算法}

\subsection{DQN}

DQN 使用神经网络近似动作价值函数 $Q(s,a)$，并从 replay buffer 中采样训练。其目标值为：
\begin{equation}
y=r+\gamma\max_{a'} Q_{\text{target}}(s',a').
\end{equation}
当前 DQN 运行使用 2000 个训练 episode、20 个测试 episode、学习率 $5\times10^{-4}$、折扣因子 $\gamma=0.99$、replay buffer 大小 10000、batch size 64，以及软更新系数 $10^{-3}$。

\subsection{Double DQN}

Double DQN 用在线网络选择下一步动作，用目标网络评估该动作价值：
\begin{equation}
a^*=\arg\max_{a'} Q_{\text{online}}(s',a'),
\end{equation}
\begin{equation}
y=r+\gamma Q_{\text{target}}(s',a^*).
\end{equation}
当前 Double DQN 与 DQN 使用相同的主要超参数，只是通过 seed offset 区分随机流。

\subsection{PPO}

PPO 使用带裁剪目标的 policy gradient 同时训练随机策略和价值函数。当前 PPO 运行使用学习率 $10^{-4}$、折扣因子 0.99、GAE 参数 0.95、clip range 0.15、entropy coefficient 0.03、4 个 update epochs、minibatch size 64 和 reward scaling 0.01。引入 PPO 的原因是 Beer Game 的订货策略可以直接通过随机策略优化，而不完全依赖 DQN 中可能较噪声的价值估计。

\subsection{高维动作智能体}

高维动作设置将一维订货量替换为三维订单向量。每一维可以理解为一个订货渠道或供应商。每个通道的可选订货水平为 $\{0,5,10,15,20\}$，总订货量被限制在 5 到 20 之间。过滤非法组合后，动作空间包含 34 个离散组合动作。DQN 和 Double DQN 在这些组合动作上学习 Q 值；High-Dim PPO 使用相同的动作枚举，但训练 categorical actor-critic 策略，并在环境交互前将选中的类别映射回订单向量。

\section{讨论}

当前最重要的结论是：标准动作方法整体仍强于高维动作方法。在刷新后的对比中，Double DQN 具有最高标准动作平均收益，而 PPO 具有最低方差和最高标准动作需求满足率。PPO 的优势并不是来自明显更高的平均订货量，因为其平均订货量仍与 DQN 和 Double DQN 接近；更合理的解释是它在库存保护和库存持有成本之间取得了更好的平衡。

Double DQN 在当前一维动作空间中平均收益略高于 DQN，并且将测试收益标准差从 67.19 降到 48.77。它还使用了更低平均库存和平均订货量，因此其价值主要体现为风险降低和更精简的库存行为，而不是大幅提升收益。

高维动作建模目前仍没有超过标准动作智能体。虽然它可以表达更丰富的订单分配策略，但组合动作空间增大了探索难度，也提高了价值估计或策略搜索方差。在刷新后的对比中，High-Dim Double DQN 将高维场景下的最佳平均收益提升到 445.50。High-Dim PPO 没有超过该平均收益，但它具有最低的高维收益方差，说明 policy gradient 仍然是有价值的方向。总体来看，高维方法与标准动作方法之间仍有明显差距，当前枚举式向量动作设计尚未学到优于简单一维订货的策略。

\section{后续建议}

后续训练应优先做三件事。第一，对每个算法使用多个随机种子重复实验，并报告置信区间；当前结果仍是单次运行对比，不能作为最终统计结论。第二，将 PPO 作为主要标准动作 baseline，重点调节 entropy coefficient、clip range 和 reward scaling。第三，继续推进高维 PPO 方向，但应避免将每个组合订单都视为互不相关的 categorical 动作。更合适的方式可能是使用 factorized policy 或 autoregressive order allocator，以更好匹配多渠道订货结构并降低探索难度。

实验管理方面，当前 Hydra 结构应该保留。每次运行已经能够在 \texttt{results/<algorithm>/} 下保存运行时间、解析后配置、日志、模型、训练分数、测试分数和 summary。报告中应继续明确列出使用了哪些运行目录，因为绘图脚本在使用算法名称时默认选择该算法最新的完整运行；如果需要复现实验图，应直接指定结果路径或记录 target list。

\section{结论}

根据当前保存的实验结果，标准动作智能体仍是 Beer Game 库存控制任务中表现最好的策略。Double DQN 获得最高标准动作平均收益 631.92；PPO 获得最低标准动作收益标准差 39.30 和最高标准动作需求满足率 0.934。在高维动作空间中，High-Dim Double DQN 获得最高平均收益 445.50；High-Dim PPO 平均收益为 406.50，并且收益波动低于高维价值函数方法。不过，高维策略整体仍低于标准动作策略，因此下一步应进行多随机种子评估，并重点优化因子化或自回归式高维动作策略。
